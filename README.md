      <td>
        <h3>Prerequisites</h3>
        <ul>
          <li>Android Studio Hedgehog (2023.1.1) or newer</li>
          <li>JDK 11 or higher</li>
          <li>Android SDK 35 (Android 15)</li>
          <li>Minimum Android 12.0 (API level 32)</li>
          <li>Firebase account with Firestore enabled</li>
          <li>Git for repository cloning</li>
          <li>Device with at least 2GB RAM recommended</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>
        <h3>Firebase Configuration</h3>
        <ol>
          <li>
            <strong>Create Firebase Project</strong>
            <ul>
              <li>Visit <a href="https://console.firebase.google.com/">Firebase Console</a></li>
              <li>Create a new project or select existing one</li>
              <li>Enable Google Analytics (recommended)</li>
            </ul>
          </li>
          <li>
            <strong>Configure Android App</strong>
            <ul>
              <li>Add Android app with package name: <code>vcmsa.projects.prog3c</code></li>
              <li>Download <code>google-services.json</code> configuration file</li>
              <li>Place file in <code>app/</code> directory of the project</li>
            </ul>
          </li>
          <li>
            <strong>Enable Firebase Services</strong>
            <ul>
              <li>Authentication: Enable Email/Password provider</li>
              <li>Firestore Database: Create in production mode</li>
              <li>Storage: Configure for image uploads</li>
              <li>Security Rules: Set up user-specific access controls</li>
            </ul>
          </li>
        </ol>
      </td>
    </tr>
    <tr>
      <td>
        <h3>Development Setup</h3>
        <ol>
          <li>
            <strong>Clone Repository</strong>

              <code>cd Prog3C_POE</code>
            </div>
          </li>
          <li>
            <strong>Open in Android Studio</strong>
            <ul>
              <li>Launch Android Studio</li>
              <li>Select "Open an existing project"</li>
              <li>Navigate to cloned repository</li>
              <li>Wait for Gradle sync completion</li>
            </ul>
          </li>
          <li>
            <strong>Configure Dependencies</strong>
            <ul>
              <li>Ensure all Firebase dependencies are up to date</li>
              <li>Verify MPAndroidChart library integration</li>
              <li>Check AmbilWarna color picker dependency</li>
              <li>Confirm Material Design 3.0 components</li>

          </li>
